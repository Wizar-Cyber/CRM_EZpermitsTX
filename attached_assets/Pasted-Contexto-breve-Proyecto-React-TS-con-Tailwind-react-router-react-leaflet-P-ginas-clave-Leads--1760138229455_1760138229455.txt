🧱 Esquema de tabla (Postgres)
CREATE TABLE IF NOT EXISTS routes(
  id SERIAL PRIMARY KEY,
  name TEXT NOT NULL,
  created_by TEXT NOT NULL,
  scheduled_on TIMESTAMP WITHOUT TIME ZONE NOT NULL DEFAULT now(),
  created_at TIMESTAMP WITHOUT TIME ZONE NOT NULL DEFAULT now(),
  points JSONB NOT NULL,
  route  JSONB
);

🧵 Endpoints (contrato)

POST /api/routes → body { name, created_by, points:[{id,address,lat,lng,case_number?}] } → calcula route con ORS → guarda → { success, data }.

PUT /api/routes/:id → mismo body → recalcula route si cambian points.

DELETE /api/routes/:id → 200/204.

GET /api/routes → { data:[{id,name,created_by,created_at,scheduled_on}] }.

GET /api/routes/:id → fila completa con points y route.
